# 创建多语言电子邮件或SMS投放

Adobe Campaign使您能够简化发送多语言活动的操作复杂性。 电子邮件投放和SMS投放均可设置为多语言，这样您就可以根据用户档案喜欢的语言发送电子邮件或SMS。

此视频介绍如何基于电子邮件新闻快讯示例中的模板创建多语言投放。视频中演示的方法适用于电子邮件和短信投放。
