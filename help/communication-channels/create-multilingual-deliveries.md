# 创建多语言电子邮件或短信投放

Adobe Campaign使您能够简化发送多语言营销活动的操作复杂性。 电子邮件投放和短信投放均可设置为多语言，从而允许您根据用户档案的首选语言发送电子邮件或短信。

此视频介绍如何根据电子邮件新闻稿示例中的模板创建多语言投放。视频中演示的方法适用于电子邮件和短信投放。
