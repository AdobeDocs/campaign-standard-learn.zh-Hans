# 根据用户的 [!DNL Campaign] 个人资料

您可以使用Adobe Campaign Standard中提供的CRM配置文件属性，定位已通过PII（已知的移动设备应用程序用户）登录应用程序并显示个性化消息的用户。 此模板支持跨渠道编排用例，其中您已经根据客户的响应定向其他渠道（如电子邮件或推送）的客户，并且您希望通过应用程序内消息再次吸引这些客户。
