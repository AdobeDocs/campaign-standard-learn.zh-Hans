# 创建和编辑登陆页 {#create-and-edit-a-landing-page}

Adobe Campaign允许您构建在线表单登陆页，这些表单可用于捕获受众信息、优惠订阅服务、显示数据和扩展数据库。 登陆页还可用于获取或更新现有用户档案。
