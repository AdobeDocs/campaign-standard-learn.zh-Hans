# 根据自定义用户档案属性创建报告

Adobe Campaign的 [!UICONTROL Dynamic Reports] 设计是一种灵活的自由形式环境，您可以通过包含自定义用户档案属性来浏览数据。 它允许您通过无限制的细分和比较，将维度、指标和时间范围组合到任何组合中。

此视频介绍如何根据自定义用户档案属性创建报表。
