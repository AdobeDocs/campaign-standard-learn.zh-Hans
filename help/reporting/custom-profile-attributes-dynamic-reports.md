# 根据自定义用户档案属性创建报告

Adobe Campaign的 [!UICONTROL Dynamic Reports] 是一个灵活的自由格式环境，您可以在其中通过包含自定义配置文件属性来浏览数据。 它允许您以任意组合方式组合维度、量度和时间范围，并且允许进行无限量的细分和比较。
