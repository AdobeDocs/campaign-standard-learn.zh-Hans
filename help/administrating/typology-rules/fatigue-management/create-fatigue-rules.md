# 创建 [!UICONTROL fatigue rules]

[!UICONTROL Fatigue rules] 允许营销人员设置全局跨渠道业务规则，以自动从营销活动中排除发送过量信息的用户档案。
创建 [!UICONTROL fatigue rules]，则需要定义每个用户档案的最大消息数，并选择应用规则的时段。 在投放准备期间，会根据已发送给用户档案的消息数量，将符合条件的用户档案从投放中排除。
