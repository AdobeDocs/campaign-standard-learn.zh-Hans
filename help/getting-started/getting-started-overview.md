# Adobe Campaign Standard入门

Adobe Campaign Standard 是一款功能强大的跨渠道活动管理工具，可帮助您调整线上和线下战略以创建个性化的客户体验。

控制创建、执行和跟踪跨渠道活动的整个过程，以及根据客户的用户档案、行为、偏好和需求发送个性化消息。您可以轻松地编排、建模和自动化整个营销流程，并与其他Adobe解决方案集成。
