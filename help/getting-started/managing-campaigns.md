# 管理活动

Adobe Campaign允许您规划市场营销活动，在这些活动中，您可以创建和管理不同类型的： 电子邮件、短信、 [!UICONTROL push notifications]工作流、登陆页。 这些活动及其内容可以收集成项目。 此视频将向您展示如何在Adobe Campaign中创建和管理活动。
