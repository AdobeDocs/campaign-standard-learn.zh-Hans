# 创建测试用户档案 {#creating-a-test-profile}

利用测试用户档案可定向不符合所规定定向标准的其他收件人。消息的受众中会添加这些参数，以检测收件人数据库是否用于任何欺诈行为，或确保电子邮件送达收件箱中。
